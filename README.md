# bomcheck

料号检测系统，提供以下功能：

- 识别并处理失效料号，自动填写替换料号和描述，并统计替换情况。
- 自动判定BOM中的数量列，统计绑定料号项目的必备料号并识别缺失数量。
- 根据重要物料库输出关键信息，并在Excel中生成执行结果与剩余物料工作表。
- 提供图形界面，可选择Excel执行文件，查看结果，并编辑绑定料号系统库（支持导入导出Excel）。

## 快速开始

1. 安装依赖：

   ```bash
   pip install -r requirements.txt
   ```

2. 运行程序：

   ```bash
   python main.py
   ```

3. 在界面中：
   - 选择需要检查的BOM Excel文件。
   - 确认或调整失效料号数据库、绑定料号库以及重要物料库路径。
   - 点击“执行”后程序会直接在Excel中写入结果，并在界面显示统计信息。
   - 使用“编辑绑定料号库”可对库进行新增、编辑、删除，以及Excel导入导出操作。

执行完成后，Excel文件会新增“执行结果”和“剩余物料”两个工作表，方便复核统计数据。
