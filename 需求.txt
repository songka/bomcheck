料号检测系统：

界面：
可选择Excel文件，进行执行。下面有结果显示框，显示最后执行结果，完全没问题为绿色背景，否者为红色
显示结果：
1失效料号数量，已替换数量
2统计显示每个绑定料号的项目描述和数量（只显示有的），如果有缺失请醒目显示总共缺失的物料的料号，描述和数量
3显示重要物料数量（只显示有的）
4绑定料号系统库编辑页面

文件结构：
1.执行文件
2.配置文件：设置数据库文件名称和地址（可为文件根目录）
3.失效料号数据库：为Excel档，第1、2列记录失效料号的料号和描述，3、4列记录替换料号的料号和描述
4.绑定料号系统库：为js档，格式如下：
[
  {
    "projectDesc": "string",         // 项目描述，例如“汇川400W带刹车电机”
    "indexPartNo": "string",         // 索引料号（项目主料号）
    "indexPartDesc": "string",       // 索引料号的详细描述
    "requiredGroups": [              // 必备料号分组（每个分组是一类功能模块）
      {
        "groupName": "string",       // 分组名称，例如“驱动器”、“动力线”等
        "number": "string|number",   // [可选] 数量，例如 "4" 表示需要4个
        "choices": [                 // 多个可选料号（或选项）
          {
            "partNo": "string",                // 料号
            "desc": "string",                  // 料号描述（可以包含分类、品牌、参数等）
            "conditionMode": "string",         // [可选] 条件模式："ALL"|"ANY"|"NOTANY"
            "conditionPartNos": ["string"],    // [可选] 条件中涉及的料号列表
            "number": "string|number"          // [可选] 数量（如果单个选择有独立数量）
          }
        ]
      }
    ]
  }
]
5.重要物料描述库：为txt档案，按行分，每行一个

功能：
一、按下执行后，逐步进行以下步骤：
①在选择的Excel里面查找失效料号数据库第一列的失效料号，如果有，就将该行现有内容全部涂黑，并在后面放入对应的替换料号和描述（如果有），最后统计查找的行数量和已替换的数量
②查找选择的Excel，确认那一列为数量：先排除前3列，看都是数字，并且小数最少的那列就是数量列
③根据绑定料号系统库内“项目主料号”进行查询（避开已被成功替换的料号，使用替换后的料号搜索），统计必备料号数量（注意有些料号为多个可选，有的是需要满足其他条件才需要，如果没有数量就默认是1）
④根据必备料号进行查找，并计算数量，请注意多选项，计算出是否缺少必备料号，并根据组合算出最少缺少的组合
⑤根据重要物料描述库，查询重要物料，注意简繁体要自动转换。
⑥界面显示，在Excel新建2个工作表，一个为执行结果和统计结果，一个为统计后剩余物料列表
二、绑定料号系统库可以进行编辑，新增删除，导入和导出为Excel，注意编辑时要能方便视图，并且可以复制。
